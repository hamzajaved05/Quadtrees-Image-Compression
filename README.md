# Quadtrees-Image-Compression
Image Compression Using Quadtreess

```
  usage: main.py [-h] [--img IMG] [--threshold THRESHOLD] [--plot_edges PLOT_EDGES] [--save SAVE] [--plot PLOT]

  Convert the image to Quadtrees and back

  optional arguments:
    -h, --help              show this help message and exit
    --img IMG               Path to the image
    --threshold THRESHOLD   Threshold for Quadtrees
    --plot_edges PLOT_EDGES Plot edgges to visualize the quadtree structure | 0 for False else True
    --save SAVE             Save the output into jpeg file | 0 for False else True
    --plot PLOT             Plot the output | 0 for False else True
```
